# Personal Portfolio Website

This is my personal portfolio, it showcases my projects, background, resume, and contact information.

## Features

- Clean and responsive design for desktop and mobile
- About Me section with an introduction
- Career Path section highlighting professional growth
- Projects section with links to live demos
- Contact form and resume link
- Smooth scrolling and basic interactivity using JavaScript

## Technologies Used

- HTML5  
- CSS3  
- JavaScript
